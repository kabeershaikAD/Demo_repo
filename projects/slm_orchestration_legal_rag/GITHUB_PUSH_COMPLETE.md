## 📝 For Users Cloning Your Repo

### Setup Instructions:

1. **Clone repository:**
   ```bash
   git clone https://github.com/kabeershaikAD/Demo_repo.git
   cd Demo_repo/projects/slm_orchestration_legal_rag
   ```

2. **Create config.py:**
   ```bash
   cp config.example.py config.py
   ```

3. **Add API keys:**
   - Edit `config.py`
   - Replace `YOUR_GROQ_API_KEY_HERE` with your actual key
   - Replace `YOUR_OPENAI_API_KEY_HERE` with your actual key (optional)

4. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

5. **Run the application:**
   ```bash
   streamlit run slm_orchestration_app.py
   ```
