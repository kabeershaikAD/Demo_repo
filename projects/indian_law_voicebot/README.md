# 🎤 Indian Law Voicebot

## Overview

Voice-based legal assistant for Indian law queries.

## Project Structure

```
indian_law_voicebot/
├── workingone.py          # Main application
├── law_buddy.db          # Database
├── chroma_db_/           # Vector database
├── evaluation/           # Evaluation files
└── ground_truths.csv     # Test data
```

## Quick Start

```bash
# Navigate to project
cd projects/indian_law_voicebot

# Install dependencies
pip install -r requirements.txt

# Run the application
python workingone.py
```
