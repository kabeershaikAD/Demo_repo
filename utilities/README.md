# 🛠️ Utilities

## Overview

Utility scripts for database management, data loading, and demos.

## Project Structure

```
utilities/
├── database/                   # Database utilities
│   ├── check_database.py
│   ├── load_chroma.py
│   └── load_kaggle_data.py
├── data_loading/               # Data loading scripts
│   ├── data_loader.py
│   ├── ingest.py
│   └── embed.py
└── demos/                      # Demo scripts
    ├── hod_demo.py
    └── demo_system.py
```

## Usage

These utilities support the main projects:
- **SLM Orchestration**: Uses data loading utilities
- **Database Builders**: Uses database utilities
- **Demos**: Presentation and demonstration scripts
